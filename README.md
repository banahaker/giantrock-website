# 巨磐資安培訓學院 GiantRock Cybersecurity Academy

專為台灣中小製造業打造的實戰型資安培訓學院官方網站

## 🏗️ 專案技術

- **前端框架**: React 19
- **建置工具**: Vite 7
- **路由管理**: React Router DOM 7
- **包管理器**: pnpm
- **樣式**: CSS Modules

## 🚀 快速開始

### 安裝依賴
```bash
pnpm install
```

### 開發模式
```bash
pnpm run dev
```
網站將在 `http://localhost:5173` (或其他可用端口) 啟動

### 建置生產版本
```bash
pnpm run build
```

### 預覽生產版本
```bash
pnpm run preview
```

### 📚 課程體系 (/courses)
- **基礎必修線**：員工資安意識、釣魚郵件、USB防護等
- **製造業專線(OT)**：產線電腦、工控系統、外包商管理
- **合規導向線**：ISO 27001、供應鏈資安、稽核準備
- **高階主管專班**：董事會課程、決策模擬、危機處理

- 公司資訊

## 🎯 核心特色

1. **實戰導向** - 不講理論，只教可執行的防護措施
2. **深耕製造業** - 理解產線與工控系統的實際需求
3. **成本思維** - 符合中小企業預算的解決方案
4. **合規專業** - 熟悉稽核要求與法規標準

## 📝 核心定位

**專為台灣中小製造業打造的實戰型資安培訓學院**

- 勒索病毒防線，從員工開始
- 讓資安變成競標優勢，而不是成本
- 深耕製造業實務，了解產線需求
